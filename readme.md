# HCS project

## running environment
    Java 1.8 - 17
    MacOS 13+

## running process
    加载Maven后直接运行KeyPressed. 
    开始输入密码（不一定需要再命令行中，键盘全局监控），输入完成后按enter键会有提示
    需要按住X键周围的Y个键Z秒。如果达到要求，输出成功，否则提示失败原因。
    失败重复
